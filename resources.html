Resources

These resources are what I used during the development of this website and I'd recommend all of them.

Atom

When I began coding I found myself constantly installing new IDE's to keep up with a vast amount of programming languages. When I found Atom I was amazed at how simple and great the interface looked. It's biggest advantage is the community built packages, instead of downloading an IDE you are able to download a package to run your code. The automatic typing suggestions provided by Atom greatly increase the productivity. I do not see myself using another code editor for a long time.

Dropbox

Dropbox is one of the best services I have ever used. Incredible file synchronization, simple to use and reliable. Coding on the go is a breeze, as soon as you save the file is synced and you're ready to continue working on any other machine. You'll never lose sleep over your harddrive corrupting. Version control allows you to reverse that huge bug you just introduced into your code. Brilliant service. My only complaint would be that restoring a version is very cumbersome.

Bootstrap

I remember coding up my first website and realising how slow and inefficient it was building a website from scratch. This website was something I made to further my understanding of Bootstraps functionality. It's great to see open-source frameworks stopping developers from having to reinvent the wheel. Bootstraps grid system is incredibly powerful, a websites layout can be accomplished within a couple minutes.

jQuery

JavaScript has always proved difficult for me to learn. Thankfully the jQuery library makes it incredibly easy to manipulate the whole page. Implementation of features such as the Parallax effect and fade in/out effects proved to be a breeze. Basic animation is accomplished in 1 or 2 lines without the need to write complex functions. If you can't accomplish a task with jQuery there is always people online who had the same problem and most likely solved it.
